# Cosimo Benvenuti — Portfolio

Sito portfolio personale (Game Developer · Unity · C#).
Bilingue IT/EN, tema scuro "code editor", responsive. Sito statico: nessun build necessario.

## Struttura

- `index.html` — pagina principale
- `styles.css` — stile (tema scuro, accento cyan)
- `script.js` — switch lingua IT/EN, animazioni, menu mobile
- `assets/` — foto, copertine demo e clip video

## Pubblicazione (GitHub Pages)

1. Crea un nuovo repository su GitHub (pubblico).
2. Carica tutti i file di questa cartella, mantenendo la cartella `assets/`.
3. Settings → Pages → Source: "Deploy from a branch" → Branch: `main` / `/root` → Save.
4. Il sito sarà online su `https://<username>.github.io/<repo>/`.
